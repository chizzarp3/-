Как это работает

Этот генератор использует цветовую теорию для создания гармоничных палитр на основе выбранного вами базового цвета. Программа переводит базовый цвет в формат HSL (оттенок, насыщенность, яркость), что позволяет создавать математически точные палитры.

Вы можете выбрать один из следующих типов цветовых гармоний:

Аналоговая

Использует цвета, расположенные рядом на цветовом круге. Создает мягкие, гармоничные сочетания.

Комплементарная

Использует противоположные цвета на цветовом круге. Создает контрастные, живые сочетания.

Триадическая

Использует три цвета, равномерно расположенные на цветовом круге. Создает сбалансированные, яркие сочетания.

Тетрадическая

Использует четыре цвета, образующие прямоугольник на цветовом круге. Создает богатые, разнообразные сочетания.

Монохроматическая

Использует различные оттенки, тона и оттенки одного цвета. Создает утонченные, элегантные сочетания.

Раздельно-комплементарная

Использует основной цвет и два цвета, прилегающие к его комплементарному. Создает гармоничные контрастные сочетания.
